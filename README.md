# Laiba's Professional Portfolio 🚀

A modern, responsive, and beautiful portfolio website built with **Next.js** and **Tailwind CSS**.

## 🎯 Features

✨ **Modern Design** - Clean, professional gradient interface with dark theme
📱 **Fully Responsive** - Works perfectly on desktop, tablet, and mobile devices
⚡ **Fast Performance** - Built with Next.js for optimal speed and SEO
🎨 **Customizable** - Easy to update with your information
🔗 **Social Links** - Links to GitHub, email, and other social profiles

## 📋 Portfolio Sections

- **About** - Introduction and professional summary
- **Skills** - Technologies and programming languages
- **Projects** - Showcase of completed projects with GitHub links
- **Education** - Academic achievements and certifications
- **Contact** - Easy contact methods (email, phone, social media)

## 🛠️ Technologies Used

- **Next.js 16.2.3** - React framework for production
- **TypeScript** - Type-safe JavaScript
- **Tailwind CSS** - Utility-first CSS framework
- **React 19.2.4** - JavaScript library for UI

## 📦 Installation & Setup

1. **Navigate to the portfolio directory:**
   ```bash
   cd d:\portfolio
   ```

2. **Run the development server:**
   ```bash
   npm run dev
   ```

3. **Open your browser:**
   Visit `http://localhost:3000`

## 🚀 Deployment

### Option 1: Vercel (Recommended)
1. Push code to GitHub
2. Visit [vercel.com](https://vercel.com)
3. Connect your repository and deploy

### Option 2: Build for Production
```bash
npm run build
npm run start
```

## ✏️ Customization Guide

Edit `src/app/page.tsx` to update:
- Personal information (name, email, phone)
- About section
- Skills list
- Projects with GitHub links
- Education details
- Social media links
